# Convertv2

> This project will convert a command from Mirai v1 to Mirai v2

> WIP

## How To Use

- Open cmd in this project folder.

- Type this in cmd:
```cmd
mklink /d "v1" "<Path To V1>"
```

Replace <Path To V1> with the actual path. Eg: C:\Users\SpermLord\Desktop\mirai

- Use:
```
node index <command>
```

Eg: ```node index help```
